# 🚀 DejaVu

> **“哎？这张图我是不是刚才删过？”**
>
> **别怀疑你的记忆，相信 DejaVu。**

---

### 🤔 这是一个什么鬼东西？

DejaVu 是一个跑在终端里的 TUI（文本用户界面）小工具。它的专长不是让你产生“似曾相识”的错觉，而是帮你**消灭**那些让你产生错觉的**多余文件**。

我们都知道那个痛：从网上疯狂“进货”图片和视频，结果文件夹里塞满了“同名(1)”、“新建文件夹 (2)”以及无数张长得一模一样的表情包。看着那几个 G 的重复垃圾，是不是感觉头皮发麻？

**别手动一个个翻了，那是对生命的浪费！**

把 DejaVu 扔进你的文件夹，它会像个嗅觉灵敏的猎犬一样，瞬间把那些装模作样的“克隆体”全都揪出来。它用最复古的界面（TUI），帮你解决最现代的焦虑。

### ✨ 它能干嘛？

*   **火眼金睛**：不管是改了文件名的“双胞胎”图片，还是看起来没啥区别的相似视频，DejaVu 都能给你识别出来。
*   **所见即所得**：虽然是命令行工具，但绝不枯燥！直接在终端里预览图片缩略图和视频元数据，告别盲目盲删。
*   **告别选择困难症**：遇到重复文件？简单敲几下键盘，留新舍旧，或者留旧舍新，全在你一念之间。

### 🛠️ 怎么用？

```bash
# 别想太多，直接跑它
dejavu /path/to/your/messy/folder
```

然后，看着那些重复的家伙乖乖排队等发落吧。
